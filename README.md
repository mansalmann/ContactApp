# Contact App
## Dependencies 
 * express
 * ejs
 * express-ejs-layouts
 * express-session
 * express-validator
 * fs
 * connect-flash
 * cookie-parser

# Run
```
> git clone https://github.com/mantokreng/ContactApp.git
> cd ContactApp
> npm install
> npm start
```
The application will be served on `:3000`
